# TwinSentinel-X IoT Dataset

This repository contains the dataset used for the paper

"Trust-Aware Explainable Digital Twin Framework for IoT Anomaly Detection"

Contents

- raw_dataset.csv
- Dataset description
- Feature descriptions

## Features

- Temperature
- Humidity
- CPU Usage
- Memory Usage
- Network Traffic
- Packet Loss
- Latency
- Label

Labels

0 = Normal

1 = Attack
